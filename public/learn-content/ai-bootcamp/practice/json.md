JSON არის სტრუქტურირებული ფორმატი, რომელსაც იყენებენ აპები, ავტომატიზაციები და API-ები.

AI-ს შეგიძლია სთხოვო პასუხი JSON-ად, თუ გინდა შედეგის გადატანა სხვა სისტემაში.

**მაგალითი:**

```json
{
  "task": "LinkedIn პოსტის შექმნა",
  "audience": "მცირე ბიზნესის მფლობელები",
  "tone": "პრაქტიკული"
}
```
