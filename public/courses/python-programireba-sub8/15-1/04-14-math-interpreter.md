Python - ს ჩაშენებული აქვს მათემატიკური ოპერაციები. შეგიძლია დაწერო კოდი რომლითაც მიმატება, გამოკლება, გამრავლება ან გაყოფა გამოიყენო შენს პროგრამაში და იმუშავებს. მაგრამ მოდით დავწეროთ პროგრამა რომელიც ისეთ მომხმარებელს მისცემს ამ ოპერაციების გამოყენების საშუალებას ვინც Python - ის კოდის წერა არ იცის.

**დავალება**

ფაილში სახელად interpreter.py შექმენი პროგრამა რომელიც მომხმარებელს ჰკითხავს შეიყვანოს რაიმე მათემატიკური გამოსახულება (Expression), გამოთვალის გამოსახულების შედეგს და დააბრუნებს წილადი რიცხვის ფორმატში (float), მძიმის შემდეგ 1 რიცხვის სიზუსტით.

ჩათვალე რომ მომხმარებლის მიერ შეყვანილი გამოსახულების ფორმატი ასეთია x y z. გამოყოფილი თითო ცარიელი ადგილით, სადაც:

- x მთელი რიცხვია (int)
- y რომელიმე ოპერაციაა +,-,* ან /
- z მთელი რიცხვია (int)

ასე მაგალითად - თუ მომხმარებელი შეიყვანს 1 + 1 - ს, მაშინ შენმა პროგრამამ უნდა დაუბრუნოს 2.0.

ჩათვალე რომ თუ გაყოფის ოპერაციას ვაკეთებთ, 0 - ზე გაყოფა არ განიხილება და შენმა პროგრამამ არ არის აუცილებელი გაითვალისწინოს ეგ სიტუაცია.

**დააკვირდი?** როგორც თავად python ბრძანებაა ინტერპრეტატორი Python ენით დაწერილი კოდისთვის, შენი interpreter.py - ც იქნება ინტერპრეტატორი მათემატიკური გამოსახულებებისთვის.

 

**

მინიშნებები:**

- გაიხსენე რომ str - ს საკმაოდ ბევრი სასარგებლო მეთოდი აქვს: [https://docs.python.org/3/library/stdtypes.html#string-methods](https://docs.python.org/3/library/stdtypes.html#string-methods) მათ შორის split() - იც, რომელსაც  შეუძლია სტრიქონის გახლეჩვა კონკრეტული გამყოფის მიხედვით. მიხებული ნახლეჩების თანმიმდევრობის მნიშვნელობები კი შეგიძლია სათითაოდ შეინახო ცვლადებში. მაგალითად თუ მომხმარებელი შეიყვანს 1 + 1 - ს, მაშინ ეს კოდი მოგცემს მნიშვნელობებს x,y,z ცვლადებში:
`x, y, z = expression.split(" ")`

#### სანამ დაიწყებ

1. Terminal - ში გაუშვი ბრძანება `cd` - ამ ბრძანების გაშვებით თავს დაიზღვევ რომ ნამდვილად შენს მთავარ ფოლდერში ხარ. 
2. შემდეგ ისევ Terminal - ში გაუშვი ბრძანება `mkdir interpreter` - ამ ბრძანებით შექმნი ახალ ფოლდერს სახელად interpreter შენი დავალებისთვის.
3. შემდეგ გაუშვი `cd interpreter` - ამ ბრძანებით შეხვალ შენს მიერ შექმნილ interpreter ფოლდერში. 
4. ახლა ამ ფოლდერში შექმენი ფაილი სახელად `interpreter.py`- ამ ბრძანების გამოყენებით `code interpreter.py`
5. დაიწყე დავალების შესრულება `interpreter.py` ფაილში.

#### როგორ უნდა გატესტო შენი პროგრამა?

როდესაც ჩათვლი რომ შედეგს მიაღწიე და გინდა დარწმუნდე რომ შენი პროგრამა სწორად მუშაობს, შეგიძლია გატესტო რამდენიმე გზით. 

##### გატესტე ხელით

- გაუშვი შენი პროგრამა python interpreter.py, აკრიფე 1 + 1 და დააჭირე Enter ღილაკს. შენმა პროგრამამ უნდა დაგიბრუნოს 2.0.
- გაუშვი შენი პროგრამა python interpreter.py, აკრიფე 2 - 3 და დააჭირე Enter ღილაკს. შენმა პროგრამამ უნდა დაგიბრუნოს -1.0.
- გაუშვი შენი პროგრამა python interpreter.py, აკრიფე 2 * 2 და დააჭირე Enter ღილაკს. შენმა პროგრამამ უნდა დაგიბრუნოს 4.0.
- გაუშვი შენი პროგრამა python interpreter.py, აკრიფე 50 / 5 და დააჭირე Enter ღილაკს. შენმა პროგრამამ უნდა დაგიბრუნოს 10.0.
