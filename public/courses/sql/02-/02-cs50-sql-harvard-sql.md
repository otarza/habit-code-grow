ჩვენი კურსი შედგენილია Harvard - ის ოფიციალური კურსის ინგლისური ვერსიის შესაბამისად, ამიტომ სურვილის შემთხვევაში შეგიძლიათ მისი ჰამოყენებაც: [https://cs50.harvard.edu/sql/2024/](https://cs50.harvard.edu/sql/2024/)

თუმცა გაითვალისწინეთ რომ ჰარვარდის ფორმატი განსხვავებულია და ჩვენ თუ მოკლე, მარტივად გასაგები ვიდეოები გვაქვს, იქ გრძელი 2 საათიანი ლექციებია. 

როგორც ამ კურსის ინსტრუქტორი რეკომენდაციას გიწევთ რომ ჯერ უყუროთ ჩვენი კურსის ქართულ ვერსიას და ნაბიჯ-ნაბიჯ, თქვენთვის კომფორტული ტემპით გაიაროთ ყველა ის მაგალითი და დავალება რომელიც გზადაგზა შეგხვდებათ და შემდეგ გადამეორებისთვის ინგლისური კურსის გრძელ ლექციებსაც შეხედოთ. ამ გზით გაცილებით მეტად გაგიადვილდებათ სწავლაც და გაგიმყარდებათ ნასწავლი მასალაც.
