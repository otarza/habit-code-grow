### **рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ! рЃЕрЃљрЃћрЃгрЃћрЃарЃћ рЃАрЃљрЃЏрЃћрЃюрЃбрЃЮрЃарЃЮ рЃ»рЃњрЃБрЃцрЃерЃў ­ЪЉЄ**

![](../../../media/external/images/oto-pointing-to-presentation-e1741862804111-9edd7649.jpg)

рЃњрЃўрЃюрЃЊрЃљ рЃЏрЃћрЃюрЃбрЃЮрЃарЃў рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃЮрЃА рЃњрЃљрЃБрЃњрЃћрЃЉрЃљрЃарЃў рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃЎрЃЋрЃћрЃЋрЃљрЃАрЃљ рЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃљрЃерЃў? рЃЕрЃљрЃћрЃгрЃћрЃарЃћ рЃАрЃљрЃЏрЃћрЃюрЃбрЃЮрЃарЃЮ рЃ»рЃњрЃБрЃцрЃерЃў рЃЊрЃљ рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃўрЃА рЃЏрЃћрЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃћрЃЮрЃЉрЃўрЃЌ рЃњрЃљрЃўрЃљрЃарЃћ рЃЏрЃЌрЃћрЃџрЃў рЃЎрЃБрЃарЃАрЃў: **[https://www.bitcamp.ge/courses/python-programireba-samentoro/](https://www.bitcamp.ge/courses/python-programireba-samentoro/)**

 

рЃЏрЃћрЃюрЃбрЃЮрЃарЃў рЃљрЃЏ рЃЎрЃБрЃарЃАрЃерЃў рЃњрЃљрЃюрЃ«рЃўрЃџрЃБрЃџ рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљ рЃЎрЃБрЃЌрЃ«рЃўрЃЌ рЃерЃћрЃњрЃљрЃ«рЃћрЃЊрЃћрЃЉрЃА, рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃџрЃћрЃЦрЃфрЃўрЃћрЃЉрЃА рЃЕрЃљрЃњрЃўрЃбрЃљрЃарЃћрЃЉрЃА рЃЊрЃљ Harvard - рЃўрЃА рЃАрЃћрЃарЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљрЃА рЃњрЃљрЃАрЃгрЃљрЃЋрЃџрЃўрЃА.

 

**

рЃарЃљрЃф рЃЏрЃЌрЃљрЃЋрЃљрЃарЃўрЃљ!** рЃЏрЃљрЃарЃбрЃЮ рЃљрЃа рЃЏрЃЮрЃњрЃўрЃгрЃћрЃЋрЃА рЃЎрЃБрЃарЃАрЃўрЃА рЃњрЃљрЃЋрЃџрЃљ рЃЊрЃљ рЃљрЃарЃљрЃАрЃЮрЃЊрЃћрЃА рЃўрЃњрЃарЃФрЃюрЃЮрЃЉ рЃЌрЃљрЃЋрЃА рЃњрЃљрЃЏрЃЮрЃБрЃЋрЃљрЃџ рЃАрЃўрЃбрЃБрЃљрЃфрЃўрЃљрЃерЃў. рЃарЃЮрЃфрЃљ рЃарЃљрЃЏрЃћрЃА рЃЋрЃћрЃа рЃњрЃљрЃўрЃњрЃћрЃЉ, рЃЏрЃћрЃюрЃбрЃЮрЃарЃў рЃЊрЃљ рЃ»рЃњрЃБрЃцрЃћрЃџрЃћрЃЉрЃў рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃљрЃю.
